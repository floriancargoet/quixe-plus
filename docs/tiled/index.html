A work of interactive fiction.

Demo of the QuixePlus Tilemap feature was created with Inform and has IFID 2D0EF683-25B6-4213-9965-C9172246ABD4. To play a work like this one, you need an interpreter program: many are available, among them Gargoyle for Mac OS X or Windows; Windows Frotz or Windows Glulxe for Windows. Or you can play without downloading anything by following the 'Play In-Browser' link, using the QuixePlus interpreter. You'll need to have Javascript enabled on your web browser.
